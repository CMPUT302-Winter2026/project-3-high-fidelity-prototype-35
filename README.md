## About the App

Vocabulary Explorer is a high-fidelity prototype of a Plains Cree (nêhiyawêwin) vocabulary learning application built for the University of Alberta Language Technology Lab (ALTLab). It supports two modes:

- **Novice Mode** - browse vocabulary by category, search words in Cree or English, view word details with example sentences and semantic relationships, and create and save lesson plans.
- **Expert Mode** - all Novice features plus morphological analysis (word class, stem, inflection), lexicographic notes, semantic gap analysis, part-of-speech filtering, and word data export.

## Installation & Running

### Prerequisites

- **Node.js** v18 or later
- npm (comes with Node.js)

### Steps

```bash
# 1. Navigate into the prototype folder
cd prototype

# 2. Install dependencies
npm install

# 3. Navigate into the src folder
cd src

# 3. Start the development server
npm run dev
```

Once running, the terminal will print a local URL such as:

```
VITE ready
➜  Local:   http://localhost:5173/
```

Go to the local URL to interact with the prototype.
